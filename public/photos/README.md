# Photos Folder

## So fügst du Fotos hinzu:

1. Kopiere deine Fotos aus `D:\Games\Dafz_r7\Dafz_r7\Fotos` in diesen Ordner
2. Benenne sie sinnvoll (z.B. `hero.jpg`, `bike1.jpg`, `bike2.jpg`)
3. Die Fotos können dann im Code verwendet werden mit:
   ```tsx
   <img src="/photos/hero.jpg" alt="..." />
   ```

## Empfohlene Dateinamen:

- `hero.jpg` - Hauptbild für Hero Section
- `bike1.jpg`, `bike2.jpg`, `bike3.jpg` - Bike Section Bilder
- `gallery1.jpg` bis `gallery6.jpg` - Gallery Bilder
- `lifestyle1.jpg` bis `lifestyle4.jpg` - Lifestyle Section
- `instagram1.jpg`, `instagram2.jpg`, `instagram3.jpg` - Instagram Preview

## Wichtig:
- Verwende JPG oder PNG Format
- Optimale Größe: 1920x1080px oder größer
- Dateinamen ohne Leerzeichen oder Sonderzeichen
